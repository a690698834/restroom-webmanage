## 使用

### 使用命令行

```bash
$ git clone https://github.com/yiningzeng/restroom-webmanage.git
$ cd restroom-webmanage
$ npm install
$ npm start         # 访问 http://localhost:8000

# build
$ npm run-scrpit build
```
#### npm 下载慢可以使用cnpm代替
